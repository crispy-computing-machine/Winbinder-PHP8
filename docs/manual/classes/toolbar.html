ToolBar class

A ToolBar is a horizontal bar container with buttons.

Split buttons

ToolBar buttons can now be configured as split buttons with independent primary and arrow-click behavior.

  • Use wb_toolbar_attach_split_menu($toolbar, $buttonId, $menu) to attach a dropdown menu.
  • Use wb_toolbar_set_split_default($toolbar, $buttonId, $commandId) to change the default action used by the primary click.
  • In your callback, $lparam1 is WBC_SPLIT_PRIMARY for primary clicks and WBC_SPLIT_DROPDOWN for arrow clicks.
  • For split events, $lparam2 contains the selected/default command id.

See toolbar_split_run.php for a "Run" + "Run With Options" pattern.
