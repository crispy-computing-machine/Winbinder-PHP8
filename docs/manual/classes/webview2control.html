WebView2Control class

The WebView2Control class is a modern embedded browser control intended for hybrid desktop apps.

Unlike HTMLControl (Internet Explorer engine), this control checks for WebView2 runtime availability at runtime and fails gracefully when the runtime is missing.

  • Create with wb_create_control(..., WebView2Control, ...).
  • Navigate with wb_webview2_navigate($ctrl, $url).
  • Set inline HTML with wb_webview2_set_html($ctrl, $html).
  • Execute JavaScript with wb_webview2_execute_script($ctrl, $script).
  • Dispatch script messages into callbacks with wb_webview2_dispatch_script_message($ctrl, $message).

Compatibility: existing HTMLControl APIs remain unchanged for legacy code.
