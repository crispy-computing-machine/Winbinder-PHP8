DateTimePicker class

The DateTimePicker control provides compact date/time input with optional null selection support.

Creating the control

Create it with wb_create_control() and class DateTimePicker. Use WBC_NUMBER for time-only spinner mode, and pass WBC_DTP_ISO in the param argument to enable nullable UI and ISO-formatted callback payloads.

Date and time APIs

Use the dedicated PHP APIs:

  • wb_get_datetime() / wb_set_datetime() to read or write the selected value (timestamp or ISO string).
  • wb_get_datetime_bounds() / wb_set_datetime_bounds() to manage minimum/maximum allowed values.
  • wb_get_datetime_format() / wb_set_datetime_format() to read/write the display format string (getter returns the custom format set through the API).
  • wb_get_datetime_empty() / wb_set_datetime_empty() to read/write empty (null) state when supported.

Notifications

In callbacks, DateTimePicker change notifications are dispatched through the normal control callback path. The payload is normalized as follows:

  • Default: param1 = Unix timestamp of selected date/time.
  • When created with WBC_DTP_ISO in param: param1 = compact ISO numeric value YYYYMMDDHHMMSS.
  • param2 = 1 when control is empty, otherwise 0.
