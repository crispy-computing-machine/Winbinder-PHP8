ScintillaEdit

ScintillaEdit is a source-code editing control backed by the Scintilla runtime.

Windows class

Scintilla

Description

This class provides a text editing surface suitable for code. Phase 3 adds PHP-editor setup support via wb_scintilla_apply_php_preset(), plus lexer/keyword/style configuration helpers (wb_scintilla_set_lexer(), wb_scintilla_set_keywords(), wb_scintilla_set_style()). Phase 4 adds callback notification mapping for editor events: WBC_SCN_MODIFIED, WBC_SCN_UPDATEUI, WBC_SCN_MARGINCLICK, and WBC_SCN_CHARADDED. Phase 5 adds IDE ergonomics helpers for autocomplete/calltips and visibility toggles for whitespace and EOL markers.

Runtime dependency: SciLexer.dll must be available at startup. If the runtime is missing, ScintillaEdit controls cannot be created.
