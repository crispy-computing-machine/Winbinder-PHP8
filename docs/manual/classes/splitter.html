Splitter

A composite WinBinder control that arranges two child panes separated by a draggable divider.

Create with wb_create_control() using class Splitter.

  • Use WBC_SPLIT_VERTICAL (default) for left/right panes.
  • Use WBC_SPLIT_HORIZONTAL for top/bottom panes.
  • Divider position can be controlled in pixels or ratio (0..1000).
  • Pane minimum sizes are enforced while dragging/resizing.
