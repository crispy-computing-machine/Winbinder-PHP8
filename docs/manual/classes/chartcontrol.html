ChartControl

A lightweight chart control for plotting series data in WinBinder.

Create with wb_create_control() using class ChartControl.

  • Supported types (v1): line, bar, scatter.
  • Input points accepted as associative arrays (x, y, optional label) or compact arrays [x, y].
  • String x-values are normalized to categorical indexes and preserved as labels.
  • Hover emits WBC_CHART_POINT_HOVER; click emits WBC_CHART_POINT_CLICK.

Related functions:

wb_chart_set_options
