wb_watch_path

int|false wb_watch_path (string path [, bool recursive = false [, int debounce = 200]])

Starts a Win32 directory change watcher for path. The function returns a watcher id that can be used with wb_watch_poll() and wb_unwatch_path().

Parameters

path is the directory to watch. recursive enables recursive monitoring of subdirectories. debounce sets the coalescing window (milliseconds) used to collapse bursts of duplicate notifications.

Return values

Returns an integer watcher id on success, or false on failure.
