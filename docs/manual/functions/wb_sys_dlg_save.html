wb_sys_dlg_save

string wb_sys_dlg_save (int parent [, string title [, null|string|array filter, string path[, string filename [, string default_extension]]]])

Displays the standard Save As dialog box. Returns the typed file name, if any, or a blank string if the dialog box was canceled. Returns NULL if not successful.

Parameters

parent is a handle to the WinBinder object that will serve as the parent for the dialog box.
title is the optional string to be displayed as the dialog box caption.
filter accepts NULL, a legacy filter string, or an array. If filter is NULL, the dialog box displays the default "All files" filter.

  • Extension array shorthand: ['png', '.jpg', 'txt'] (auto-normalized).
  • Explicit pair array: [['Images', '*.png;*.jpg'], ['All files', '*.*']].

path is an optional folder used to initialize the dialog box.
filename is an optional default name for the file.
default_extension is optional and sets lpstrDefExt. When filters are present, Windows generally uses the selected filter entry (and first filter by default) to decide which extension is appended.

Examples

// Extension array shorthand + default filename/ext

$filename = wb_sys_dlg_save($window, 'Save As', ['txt', 'log'], $path, 'output', 'txt');

// Explicit [description, pattern] pairs

$filter = [
  ['Images', '*.png;*.jpg'],
  ['All files', '*.*']
];
$filename = wb_sys_dlg_save($window, 'Export', $filter, $path, 'image', 'png');
