wb_vlc_set_media

bool wb_vlc_set_media(int wbObject, string path [, bool autoplay = false])

Loads a media file or stream URL into a VLC control.

Parameters

wbObject: VLC control handle.

path: Local media path or URL supported by VLC.

autoplay: If TRUE, starts playback immediately.

Example

<?php
$vlc = wb_create_control($win, VLCControl, "", 10, 10, 640, 360);
wb_vlc_set_media($vlc, "C:/media/trailer.mp4", true);
