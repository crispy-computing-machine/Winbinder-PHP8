wb_set_theme

Sets the theme globally or for a specific window/control subtree.

bool wb_set_theme([int $wbObject,] int $theme)

$theme supports WBT_THEME_DEFAULT, WBT_THEME_LIGHT, and WBT_THEME_DARK. Unsupported OS combinations gracefully fall back to standard rendering while preserving the selected mode for owner-draw/theme-aware controls.
