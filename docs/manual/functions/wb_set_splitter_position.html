wb_set_splitter_position

bool wb_set_splitter_position(int splitter, int position [, bool ratio_mode])

Sets divider position for a Splitter.

If ratio_mode is TRUE, position is interpreted as a ratio in the 0..1000 range. Otherwise, the value is interpreted as pixels from the top/left pane.

Example

<?php
// 35% split
wb_set_splitter_position($splitter, 350, true);

// explicit 280px split
wb_set_splitter_position($splitter, 280, false);
