wb_sys_dlg_open

string|array wb_sys_dlg_open (int parent [, string title [, null|string|array filter, string path[, int style]]])

Displays the standard Open dialog box. Returns the selected file name (single select), an array of full file names (multiselect), or a blank string if the dialog box was canceled. Returns NULL if not successful.

Parameters

parent is a handle to the WinBinder object that will serve as the parent for the dialog box.
title is the string to be displayed as the dialog box caption.
filter accepts NULL, a legacy filter string, or an array. If filter is NULL, the dialog box uses a default "All files (*.*)" filter.

  • Legacy string: passed through for backward compatibility.
  • Extension array shorthand: ['png', '.jpg', 'txt'] (auto-normalized to wildcard patterns).
  • Explicit pair array: [['Images', '*.png;*.jpg'], ['All files', '*.*']].

path is an optional folder used to initialize the dialog box.
style is optional and defaults to 0. Use WBC_MULTISELECT to enable multiselect, which makes the function return an array.

Examples

// Extension array shorthand

$filename = wb_sys_dlg_open($window, 'Open image', ['png', '.jpg', 'txt'], dirname(__FILE__));

// Explicit [description, pattern] pairs

$filter = [
  ['Images', '*.png;*.jpg'],
  ['Text', '*.txt'],
  ['All files', '*.*']
];
$filename = wb_sys_dlg_open($window, 'Get file', $filter, dirname(__FILE__));

// Multiselect (returns array on success)

$files = wb_sys_dlg_open($window, 'Pick files', $filter, dirname(__FILE__), WBC_MULTISELECT);
