wb_set_chart_data

bool wb_set_chart_data(int chart, array xData, array yData [, int chartType = WBC_CHART_LINE])

Sets data for a Chart control created with wb_create_control().

Parameters

chart: chart control handle.

xData, yData: equal-length numeric arrays.

chartType: WBC_CHART_LINE, WBC_CHART_BAR, or WBC_CHART_SCATTER.

Behavior

The chart repaints immediately. Hovering points/bars displays a tooltip in the format: xAxisLabel: x\nyAxisLabel: y.

Example

$chart = wb_create_control($win, Chart, array("Month", "Sales"), 20, 20, 400, 240);
wb_set_chart_data($chart, array(1,2,3,4), array(12.5, 16.0, 9.8, 20.1), WBC_CHART_BAR);
wb_refresh($chart);
