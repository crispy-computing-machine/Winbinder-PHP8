wb_set_chart_data

bool wb_set_chart_data(int chart, array xData, array yData [, int chartType = WBC_CHART_LINE [, bool showAverage = null]])

Assigns X/Y data to a Chart control and redraws it.

Parameters

chart: Handle returned by wb_create_control() for a Chart control.

xData: Array of numeric X values.

yData: Array of numeric Y values. Must be the same length as xData.

chartType: One of WBC_CHART_LINE, WBC_CHART_BAR, or WBC_CHART_SCATTER.

showAverage: Optional flag that enables/disables an average guide line (when supported).

Return value

Returns TRUE on success, FALSE on failure.

Example

<?php
$win = wb_create_window(NULL, AppWindow, "Chart demo", 100, 100, 520, 360);
$chart = wb_create_control($win, Chart, array("Month", "Sales"), 20, 20, 470, 260);

wb_set_chart_data(
    $chart,
    array(1, 2, 3, 4, 5, 6),
    array(12.5, 16.0, 9.8, 20.1, 18.2, 22.0),
    WBC_CHART_BAR,
    true
);

wb_main_loop();
