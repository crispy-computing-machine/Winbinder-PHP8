wb_watch_poll

array wb_watch_poll ([int timeout = 0 [, int maxEvents = 0]])

Polls all active directory watchers and returns accumulated events. This can be called from timer callbacks, application loops, or auto-reload routines.

Event fields

Each returned event is an array with:

watch_id (watcher id), event (WBE_* constant), tick (GetTickCount timestamp), base_path (watched directory), and path (relative changed file path).

Supported event constants

WBE_FILE_CREATED, WBE_FILE_MODIFIED, WBE_FILE_DELETED, WBE_FILE_RENAMED_OLD, WBE_FILE_RENAMED_NEW.
