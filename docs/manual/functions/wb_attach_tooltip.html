wb_attach_tooltip

bool wb_attach_tooltip (int wbobject, string text [, bool show_now = false])

Attaches or updates tooltip text for any control or window. Pass NULL or an empty string to keep an empty tooltip registered for later balloon usage.
Optional show_now keeps backward-compatible hover behavior by default (false). When set to true and text is non-empty, a balloon is shown immediately and anchored to the target control.
