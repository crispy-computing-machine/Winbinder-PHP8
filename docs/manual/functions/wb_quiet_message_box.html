wb_quiet_message_box

mixed wb_quiet_message_box (int parent, string message [, string title [, int style [, int timeout]]])

Creates and displays a message box like wb_message_box, but closes it automatically after timeout milliseconds.

Parameters

parent, message, title and style behave like wb_message_box.

timeout is the auto-close interval in milliseconds. A value of 0 disables auto-close and behaves exactly like wb_message_box.

Return values

Return mapping is the same as wb_message_box:

  • TRUE for OK/Yes/Retry.
  • 0 for No.
  • FALSE for Cancel/Abort/Ignore and other negative answers.
  • NULL on error.

If the box closes by timeout, this function returns IDTIMEOUT (32000).

Example

$ret = wb_quiet_message_box($window,
    "Operation completed.",
    APPNAME,
    WBC_INFO,
    2500);

if ($ret === IDTIMEOUT) {
    // Message closed automatically
}
