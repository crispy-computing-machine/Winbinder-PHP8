wb_set_splitter_minsize

bool wb_set_splitter_minsize(int splitter, int min_pane1, int min_pane2)

Defines the minimum pane sizes in pixels for a Splitter control so users cannot drag the divider past usable bounds.

Parameters

splitter: Splitter handle.

min_pane1: Minimum size for pane 1.

min_pane2: Minimum size for pane 2.

Example

<?php
wb_set_splitter_minsize($splitter, 180, 220);
